```python # Python script to calculate prime numbers up to 500 def calculate_primes(limit): primes = [] for num in range(2, limit + 1): is_prime = True for i in range(2, int(num ** 0.5) + 1): if num % i == 0: is_prime = False break if is_prime: primes.append(num) return primes # Calculate prime numbers up to 500 prime_numbers = calculate_primes(500) print(prime_numbers) ```
